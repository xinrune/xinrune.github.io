寄存器

通用寄存器

16bit的通用，数据寄存器：AX，BX，CX，DX

为了兼容，又分8bit的寄存器height和low：
AX = AH + AL
BX = BH + BL
CX = CH + CL
DX = DH + DL

地址寄存器：

段地址寄存器：DS，ES，SS，CS
偏移地址寄存器：SP，DP，SI，DI，IP
基础地址 = 段地址 x 16（10H）
物理地址 = 基础地址 + 偏移地址

数据类型

1byte = 8bit
字节型数据 =1byte
字型数据 =2byte
